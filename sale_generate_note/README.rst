==================
Sale Generate Note
==================

This module automatically generates internal notes and mail message chatter when a sales order is confirmed

Key Features
============
* Automatically create internal notes and mail chatter on a sale order.
* Simple Configuration.
* Fully compatible with Odoo version 18.0

Why use this module?
====================
* Easy to use and lightweigth with minimal configuration required.
* Increase workflow efficiency.
* Reduces input manually.

=============
Configuration
=============

To configuration this module, you need to:
1. Go to *Settings* > *Sales Option App* > *Generate Note* > *Enable Generate Note*.
2. Create sale orders to use this features.

**Note**:
* If you do not enable configuration for this module, automatically all internal notes sale order will be disappear.
* Once activated, internal notes are automatically created for all confirmed sales orders.
